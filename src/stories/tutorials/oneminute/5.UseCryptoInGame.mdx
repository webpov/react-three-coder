How to use a crypto-currency in a 3d web game with react 
first download our 3d game template from our previous tutorial
then install the Web3.js library with npm install @solana/web3.js to the blockchain.
In your game container component file, import and set up the blockchain connection, with : const connection = new web3.Connection(web3.clusterApiUrl('devnet'));.
create a button for the wallet connection
now Add a function to read the players' wallet balance.
and then make your game by requiring tokens for special features, for example
Test your integration and make any necessary personal adjustments
Congratulations! Your game now supports Web3 functionalities
